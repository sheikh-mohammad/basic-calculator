# Basic Calculator

## Core Functionality:

1. Mathematical operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
2. Handles integer and floating-point arithmetic
3. High precision for decimal calculations
4. BODMAS Order of Operations

## Features:

1. **Input Methods:**
  - Keyboard input (CLI) -> Phase 1
  - Click buttons and keyboard input (both) -> Phase 2
2. **Display:** 
  - Current input
  - Previous Calculation history (optional)
3. **Special Functions:**
  - Clear functions (C, CE)
4. **Error Handling:**
  - Invalid input handling
  - Handle Division by zero error
  - Handle Overflow
  - Invalid operations
  
## Technical Constraints:

1. **Frontend:**
  - HTML, CSS, JS, and Tailwind CSS
2. **Backend:**
  - Python backend for complex calculations with type hints
3. **Package Management:**
  - uv for fast dependency resolution and installation
4. **Platform:**
  - Mobile responsiveness
5. **Deployment:**
  - Vercel CLI
6. **Testing:**
  - Unit tests for frontend(jest or vitest) and backend(pytest)
  
## User Experience:

1. **Target Users:**
  - General Users
2. **Interface Style:**
  - Minimalistic design
